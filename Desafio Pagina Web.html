O que é HTML?

Sobre o HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto") é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML.

  • Foi inventado em 1990
  • Tim Berners-Lee, nascido em UK em 8 de junho de 1955
  • HTML é uma linguagem para publicação de conteúdo direcionados na web.
  • HTML5 nasceu com objetivo de manipulação do elemento, possibilitando, assim, que o desenvolvedor pudesse modificar as características dos objetos de forma clara, objetiva e de maneira que fosse transparente para o usuário final, em que a última ponta do trabalho não fosse prejudicada.
  • HTML5 não oferece suporte nativo a áudio e vídeo.

Tags

Uma tag é um conjunto dos caracteres <, /, > mais uma nomenclatura. Tags HTML também podem ser definidas como palavras-chave ocultas em uma página da web que definem como o navegador deve formatar e exibir o conteúdo. A maioria das tags deve ter duas partes, uma de abertura e uma de fechamento.

HTML

Essa tag define o documento como página da web e identifica o início e o fim do documento HTML.


HEAD

O cabeçalho contém informações sobre o documento que não aparecerão na página real, como o título do documento, o autor, a folha de estilo a ser usada.


TITLE

Define o título que aparecerá na barra de título do navegador. O title deve aparecer entre as tags.


BODY

As tags body contêm todas as informações e outro conteúdo visível na página. Todas as suas imagens, links e texto sem formatação devem estar entre elas.


INPUT

Tag especifica um campo de entrada onde o usuário pode inserir dados.


P

Principal tag de texto, compõe um parágrafo.


Img

Usada para incorporar uma imagem em uma página HTML.
